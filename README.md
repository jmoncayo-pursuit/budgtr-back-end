# Budgtr Back-End

This is the back-end server for Budgtr, a budgeting application where users can log financial transactions.

## Features

- RESTful API for CRUD operations on financial transactions.
- Local JSON database for data storage.

## Technologies Used

- Node.js
- Express.js
- CORS
